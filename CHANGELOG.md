# Changelog

All notable changes to this project will be documented in this file.

## 1.0.1

- Updated project's icon to be transparent

## 1.0.0

Initial public release of C Macro Helper

- Added the `Macro Helper: Add and Align Backslashes` command to align selected macro continuation lines into a single vertical backslash column
- Added the `Macro Helper: Finalize Multiline Macro` command to align a macro block while leaving the final line without a continuation backslash
- Added the `macroHelper.alignmentGap` setting to control the spacing between the longest macro line and the aligned backslash column
